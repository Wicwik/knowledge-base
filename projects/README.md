# projects/

One page per active project — ongoing work with a goal, a status, and open questions.

**What belongs here:** A single Markdown file per project tracking what you're trying to do, where things stand, and what's next. Not a log — just the current state. When a project finishes or goes dormant, move it to `archive/`.

**Who fills it:** You, with Claude helping update status or surface relevant wiki pages.

**Example:**

```
projects/isotropic-merging-experiments.md
# Isotropic Merging Experiments

Goal: reproduce Iso-CTS results on ViT-B-16 with 8 tasks.
Status: environment set up, running baseline.
Next: compare against TSV-M on SUN397.
```
